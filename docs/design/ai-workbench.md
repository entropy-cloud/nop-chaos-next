# AI 工作台（AI Workbench）详细设计

> 本文档描述 AI 工作台模块的交互细节。

---

## 1. 会话列表

### 1.1 位置与布局

- 位于页面左侧
- 宽度可调整

### 1.2 字段定义

| 字段         | 说明               |
| ------------ | ------------------ |
| 会话标题     | 可编辑             |
| 最近更新时间 | 最后活动时间       |
| 助手类型图标 | 当前使用的助手类型 |

### 1.3 功能要求

| 功能       | 说明                   |
| ---------- | ---------------------- |
| 切换会话   | 点击会话切换当前对话   |
| 新建会话   | 创建新的对话会话       |
| 删除会话   | 删除指定会话           |
| 重命名会话 | 编辑会话标题           |
| 搜索/筛选  | 按关键词或条件筛选会话 |

### 1.4 视觉要求

- 当前选中会话需要高亮显示
- 支持玻璃拟态效果

### 1.5 Inline Rename 合同

- 会话重命名使用受控输入，不依赖 `defaultValue` + DOM 复用保留旧值
- `Enter` 提交、`Blur` 提交、`Escape` 退出重命名模式
- 当前激活会话通过 `aria-current` 暴露状态，避免仅靠颜色高亮

---

## 2. 对话区域

### 2.1 消息列表

位于页面右侧，展示当前会话的聊天内容：

| 特性          | 说明                                         |
| ------------- | -------------------------------------------- |
| 消息区分      | 区分「用户」与「AI」消息，不同角色用不同样式 |
| Markdown 渲染 | AI 消息支持代码块、列表、粗体等              |
| 时间戳        | 每条消息显示发送时间                         |
| 历史加载      | 支持滚动加载历史消息                         |

### 2.2 底部输入区域

| 组件         | 说明                            |
| ------------ | ------------------------------- |
| 文本输入框   | 多行输入，支持 Shift+Enter 换行 |
| 发送按钮     | 点击发送消息                    |
| 停止生成按钮 | 流式输出时显示，可中断生成      |

### 2.3 流式输出

- AI 回复需要支持流式输出效果
- 实现打字机效果
- 可使用定时器分块输出文本模拟

### 2.4 侧栏伸缩合同

- 会话侧栏支持鼠标拖拽调整宽度
- resize handle 同时暴露滑块语义与键盘调节路径，避免只有 `aria-*` 没有真实行为

---

## 3. 助手类型切换

### 3.1 预置助手类型

至少 3 种预置助手类型：

| 助手类型     | 说明                   |
| ------------ | ---------------------- |
| 通用助手     | 回答一般性问题         |
| 数据分析助手 | 专注于数据分析和报表   |
| 流程设计助手 | 帮助设计和优化业务流程 |

### 3.2 视觉要求

- 每种助手类型在 UI 上要有清晰的视觉区分
- 包括：图标、颜色、描述

### 3.3 切换行为

切换助手类型时可以选择：

| 选项             | 说明                   |
| ---------------- | ---------------------- |
| 在当前会话中切换 | 保留历史消息，更换助手 |
| 开启一个新会话   | 创建新会话并使用新助手 |

---

## 4. 上下文附加开关

### 4.1 功能说明

- 提供一个显式开关
- 控制「是否附带当前页面上下文」

### 4.2 开启状态

当开关打开时：

- 界面上要有清晰标识
- 提示用户当前提问会包含哪些上下文信息
- 显示将要发送的上下文摘要预览

---

## 5. 对话操作

### 5.1 操作列表

| 操作         | 说明                   |
| ------------ | ---------------------- |
| 复制消息内容 | 复制单条消息的文本     |
| 重新生成回复 | 让 AI 重新生成当前回复 |
| 清空对话历史 | 清除当前会话的所有消息 |

---

## 6. 数据结构

### 6.1 会话（Conversation）

```typescript
interface Conversation {
  id: string;
  title: string;
  assistantType: AssistantType;
  createdAt: Date;
  updatedAt: Date;
  messages: Message[];
}
```

### 6.2 消息（Message）

```typescript
interface Message {
  id: string;
  role: 'user' | 'assistant';
  content: string;
  timestamp: Date;
}
```

### 6.3 助手配置（AssistantConfig）

```typescript
type AssistantType = 'general' | 'data-analysis' | 'flow-design';

interface AssistantConfig {
  type: AssistantType;
  name: string;
  description: string;
  icon: string;
}
```
